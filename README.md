### Step 1: Create GitHub Repository
1. Go to [GitHub](https://github.com) and create a new repository
2. Name it: `doughapp-website` (or any name you prefer)
3. Make it public
4. Don't initialize with README (we already have files)

### Step 2: Upload Files
1. Upload `index.html` and `support.html` to the repository
2. Commit and push the files

### Step 3: Enable GitHub Pages
1. Go to repository Settings
2. Scroll down to "Pages" section
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click "Save"

### Step 4: Get Your URLs
Your website will be available at:
- **Main site**: `https://yourusername.github.io/doughapp-website`
- **Support page**: `https://yourusername.github.io/doughapp-website/support.html`
